Enable your content to be easily embedded in a customisable iframe

== Description ==

This plugin is a very simple bloat free way of syndicating your content. Any post, page, or cpt or any collection of pages can be embedded on another site. Just add 'feed/lh-iframe/' to the end of the URL and you will find a simplified version of your content formatted perfectly for embedding in this way.


== Installation ==

= From your WordPress dashboard =
* Visit 'Plugins > Add New'
* Search for 'LH Iframer'
* Activate the plugin on your Plugins page
* Go to Settings->permalinks and regenerate your permalinks


== Frequently Asked Questions ==

= How do I change the ouput of this plugin? =
* Copy the template directory 'iframe-templates' to your theme or child theme directory. From there you can edit the output to customise it as needed.
